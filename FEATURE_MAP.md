# Prompt Studio Feature Map

## 文章方法论到功能映射

### 基础层

- 对话开场角色 Prompt
  - 对应页签：`角色库`
  - 交互：点击角色标签后直接预览 Prompt
  - 约束：默认中文、结论先行、严谨简洁、不编造、不确定就说明缺口
- Prompt 本质：给 AI 布置任务
  - 对应：`角色`、`任务`、`背景`、`输出格式`、`限制条件`
- 新手错误规避
  - 对应：强制用户补背景、格式、限制，不鼓励一句话裸问

### 进阶层

- 说原因，不只说要求
  - 对应页签：`说原因`
  - 输入框：`要求`、`原因`、`具体任务`、`补充限制`
- 给示例，比说十遍管用
  - 对应页签：`给示例`
  - 输入框：`任务`、`示例`、`待处理内容`、`输出要求`
- 让 AI 分步思考
  - 对应页签：`分步思考`
  - 输入框：`问题`、`背景`、`分析步骤`、`输出格式`
- 长内容放前面，问题放后面
  - 对应页签：`长文后问`
  - 输入框：`长内容`、`最后的问题`、`输出格式`、`限制`
- 指定输出格式
  - 对应页签：`指定格式`
  - 输入框：`任务`、`字段 / 模板`、`输入内容`、`格式规则`
- 告诉 AI 不要做什么
  - 对应页签：`不要做什么`
  - 输入框：`任务`、`输入内容`、`不要做的事`、`输出格式`

### 高级层

- Prompt 链
  - 对应页签：`Prompt 链`
- 让 AI 反驳自己
  - 对应页签：`反驳自己`
- 多角色视角
  - 对应页签：`多角色视角`
- 元提示
  - 对应页签：`让 AI 写 Prompt`
- 角色反转
  - 对应页签：`AI 采访我`
- 给 AI 压力
  - 对应页签：`给 AI 压力`
- 不满意继续迭代
  - 对应页签：`迭代修改`

## 当前定位

这是一个先求能用的 MVP，不是最终版产品。当前重点是把书里的方法论做成：

- 可视化输入
- 每个技巧页签都有简短说明，解释为什么用这种 Prompt 提问
- 可复用模板
- 可插入主流 AI 网站
- 可继续扩展成模板市场和团队 Prompt 资产库
